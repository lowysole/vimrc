# Installation

### 1. Clone the repository:

``
git clone https://github.com/mcapell/vimrc.git ~/.vim
``

### 2. Create symlinks:

``
ln -s ~/.vim/vimrc ~/.vimrc && ln -s ~/.vim/gvimrc ~/.gvimrc
``

### 3. Install plugins:

Open vim and type:

``
:PlugInstall
``

# Plugins:

Check `plugins.vim` file.
